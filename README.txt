How to import the database ? 

- Open your preferred DBMS (such as MySQL, MariaDB, or PostgreSQL)
- Create a new database with the name "joboffer"
- Go to the "Import" functionality of your DBMS
- Select the "joboffer" database file that you want to import.
- Choose the "joboffer" database as the target for import
- Click "Start Import" or the equivalent button in your DBMS

To change your setting for the database connexion you just have to in the program and go to "DatabaseConnexion.cs"
and you can change your settings.
